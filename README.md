# 🚦 Traffic Signal Detector

A real-time traffic signal color detection system using computer vision.
This project detects **Red, Yellow, and Green signals** from a live camera feed.

---

## ⚡ Hackathon Project

Built within **1 hour 30 minutes** during a hackathon to demonstrate rapid development and problem-solving skills.

---

## 🧠 Features

* Real-time traffic signal detection
* Detects:

  * 🔴 Red
  * 🟡 Yellow
  * 🟢 Green
* Live camera feed in browser
* Works on laptop and mobile (same network)

---

## 🛠️ Tech Stack

* Python
* OpenCV
* Flask
* HTML, CSS, JavaScript

---

## ▶️ How to Run

### 1. Clone repo

```bash
git clone https://github.com/your-username/traffic-signal-detector.git
cd traffic-signal-detector
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run app

```bash
python app.py
```

### 4. Open browser

```bash
http://127.0.0.1:5000
```

---

## 📱 Run on Mobile

* Connect phone & laptop to same WiFi
* Run app
* Find your IP using:

```bash
ipconfig
```

Open:

```bash
http://YOUR-IP:5000
```
